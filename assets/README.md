# Assets Folder

This folder should contain app icons and images. For now, the app will work without these, but Expo may show warnings.

To add proper assets later:

1. **icon.png** - App icon (1024x1024 pixels)
2. **splash.png** - Splash screen image (1242x2436 pixels)
3. **adaptive-icon.png** - Android adaptive icon (1024x1024 pixels)
4. **favicon.png** - Web favicon (48x48 pixels)

You can create simple placeholder images or use an online icon generator. The app will function without these, but they make it look more professional!
